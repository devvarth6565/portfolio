A unique, interactive portfolio website designed to replicate the classic Windows XP desktop environment. Instead of a boring scroll-down list, users interact with a simulated Operating System to discover my work.

* **Retro UI:** Pixel-perfect recreation of the Windows XP taskbar, start menu, and window management system.
* **Virtual File System:** Navigate through folders.
* **Project Discovery:** All portfolio projects are stored as files within the **"My Documents"** folder.

* **Dynamic Content:** Project data is fetched dynamically from a **PostgreSQL** database.
